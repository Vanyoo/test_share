# 划词分享 - 可编辑浮窗版 (Text Share)
一个强大的 Tampermonkey 油猴脚本，让你能够轻松地将网页上选中的文本生成精美的分享卡片。支持自定义编辑、Markdown 格式、多种配色方案以及字体选择。

## ✨ 功能特点

*   **智能划词**：支持普通网页文本及 Shadow DOM 内的文本选择。
*   **即时预览**：选中文字后点击“分享”按钮，立即生成精美的预览图片。
*   **高度自定义**：
    *   **内容编辑**：可以修改选中的文本内容，修正错别字或删减多余内容。
    *   **信息补充**：自定义“出处”和“作者”信息（默认自动获取网页标题）。
    *   **Markdown 支持**：支持 Markdown 语法渲染，让排版更丰富。
    *   **样式切换**：内置 4 种精美配色方案（浅色/深色/护眼等）。
    *   **字体选择**：自动检测并支持多种系统字体切换。
*   **高清输出**：基于 Canvas 绘制，支持 2x 倍率高清导出，文字清晰锐利。

## 🚀 安装方法
1.  **安装管理器**：首先确保你的浏览器已安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展。
2.  **安装脚本**：
    *   如果你有脚本文件，直接在 Tampermonkey 管理面板中添加新脚本并粘贴代码。
    *   或者访问脚本发布页（如果有）进行安装。
## 📖 使用指南
1.  **划选文本**：在任意网页上用鼠标划选你想要分享的一段文字。
2.  **点击分享**：选区附近会出现一个黑色的“分享”悬浮按钮，点击它。
3.  **编辑与预览**：
    *   在弹出的浮窗中，你可以看到生成的图片预览。
    *   在下方输入框修改**出处**和**作者**。
    *   勾选“允许编辑文本内容”可直接修改正文。
    *   勾选“Markdown格式”可启用 Markdown 渲染。
    *   点击底部的圆形图标切换**配色方案**。
    *   使用下拉菜单更改**字体**。
4.  **保存图片**：
    *   调整满意后，点击图片右键选择“图片另存为...”即可保存到本地。
    *   或者点击“重新生成”按钮刷新预览。
## 🛠️ 技术细节
*   **核心库**：使用 [marked.js](https://github.com/markedjs/marked) 进行 Markdown 渲染。
*   **实现方式**：原生 JavaScript + HTML5 Canvas API。
*   **兼容性**：支持现代主流浏览器（Chrome, Edge, Firefox 等）。
